第一阶段：基础环境与工具

Phase 01

目标：能熟练操作Linux命令行，理解基本概念

Linux文件系统结构

/
根目录，所有文件和目录的起点
系统的最顶层目录
/home
用户主目录，存放个人文件
/home/user/
/etc
系统配置文件目录
/etc/passwd, /etc/hosts
/var
可变数据，如日志、缓存
/var/log/
/tmp
临时文件目录，重启后清空
/tmp/temp.txt

文件操作命令

ls
列出目录内容 (list)
ls -l
cd
切换目录 (change directory)
cd /home/user
pwd
显示当前路径 (print working directory)
pwd
cp
复制文件 (copy)
cp file1 file2
mv
移动/重命名文件 (move)
mv old.txt new.txt
rm
删除文件 (remove)
rm -rf dir/
mkdir
创建目录 (make directory)
mkdir new_folder
touch
创建空文件或更新时间戳
touch newfile.txt

文件查看命令

cat
查看文件全部内容 (concatenate)
cat /etc/passwd
less
分页查看大文件
less large_file.log
head
查看文件开头部分
head -n 10 file.txt
tail
查看文件末尾部分
tail -f logfile.log
grep
搜索文件内容 (global regular expression print)
grep "error" log.txt

权限管理命令

chmod
修改文件权限 (change mode)
chmod 755 file
chown
修改文件所有者 (change owner)
chown user:group file
chgrp
修改文件所属组 (change group)
chgrp group file
权限说明
r(read)=4, w(write)=2, x(execute)=1
755 = rwxr-xr-x (所有者全部权限，其他用户读+执行)

进程管理命令

ps
查看进程状态 (process status)
ps aux
top
实时查看进程
top
kill
终止进程
kill -9 PID
htop
交互式进程查看器
htop

网络命令

ip
配置网络接口
ip addr show
ping
测试网络连通性
ping www.baidu.com
netstat
查看网络连接
netstat -tlnp
ss
查看套接字统计
ss -tlnp
curl
发送HTTP请求
curl http://example.com

文本编辑器

vim
强大的命令行编辑器
vim file.txt
基础操作: i(插入), Esc(退出插入), :wq(保存退出)
nano
简单易用的编辑器
nano file.txt
基础操作: Ctrl+O(保存), Ctrl+X(退出)

包管理命令

apt update
更新软件包列表
sudo apt update
apt install
安装软件包
sudo apt install nginx
apt remove
卸载软件包
sudo apt remove nginx

虚拟机与网络基础

VMware Workstation
常用虚拟机软件，支持多种操作系统
VirtualBox
开源虚拟机软件，免费使用
网络适配器模式
桥接模式、NAT模式、仅主机模式
子网划分
IP地址、子网掩码、网关
