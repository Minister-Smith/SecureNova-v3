第六阶段：进阶与实战

Phase 06

目标：深入内网渗透、权限维持及红队实战

内网信息收集

ipconfig
查看网络配置
ipconfig /all
netstat
查看网络连接
netstat -ano
tasklist
查看进程列表
tasklist /svc
systeminfo
查看系统信息

权限维持

计划任务
创建定时任务持久化
schtasks /create /tn "TaskName"
服务
创建恶意服务
sc create ServiceName binPath= "path"
注册表
修改注册表启动项
reg add HKLM\Software\Microsoft\Windows\CurrentVersion\Run
用户账户
创建隐藏账户
net user hidden$ password /add

横向移动

PsExec
远程执行命令
psexec \\target cmd
WMI
Windows管理规范
wmic /node:target process call create "cmd.exe"
SMB
共享文件访问
net use \\target\share
RDP
远程桌面连接
mstsc /v:target
