学习路径

循序渐进的学习旅程，从零基础到安全专家

Phase 01 基础环境与工具

学习Linux命令行操作、文件系统结构、权限管理、进程管理等基础技能

Phase 02 计算机网络

深入理解OSI七层模型、TCP/IP协议、路由原理及Wireshark抓包分析

Phase 03 Web基础

学习HTTP协议、HTML/CSS/JavaScript、数据库基础及Web开发框架

Phase 04 安全入门

掌握SQL注入、XSS、CSRF、文件上传等常见Web漏洞原理与利用

Phase 05 工具与自动化

学习Nmap、SQLmap、Metasploit等安全工具及Python自动化脚本编写

Phase 06 进阶与实战

深入内网渗透、权限维持、横向移动及红队实战技能

Phase 07 就业准备

简历优化、面试技巧、项目经验积累及职业规划指导
