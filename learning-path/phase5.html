第五阶段：工具与自动化

Phase 05

目标：掌握安全工具使用及自动化脚本编写

Nmap扫描工具

全端口扫描
扫描所有65535个端口
nmap -p- target
服务识别
识别端口上运行的服务
nmap -sV target
NSE脚本
使用Nmap脚本引擎
nmap --script vuln target
UDP扫描
扫描UDP端口
nmap -sU target

目录爆破工具

Dirsearch
快速目录扫描工具
dirsearch -u http://target
Gobuster
Go语言编写的目录爆破工具
gobuster dir -u http://target -w wordlist.txt
Wordlist
常用字典列表
SecLists目录

密码爆破工具

Hydra
多协议密码爆破工具
hydra -L users.txt -P pass.txt ssh://target
Medusa
快速并行密码爆破
medusa -h target -u user -P pass.txt -M ssh

SQLmap

基本扫描
自动检测并利用SQL注入
sqlmap -u "http://target?id=1"
获取数据库
列出所有数据库
sqlmap -u "url" --dbs
获取表
列出指定数据库的表
sqlmap -u "url" -D dbname --tables
获取数据
导出表中数据
sqlmap -u "url" -D dbname -T users --dump

Metasploit框架

启动msfconsole
启动Metasploit控制台
msfconsole
搜索漏洞
查找相关漏洞模块
search ms17-010
使用模块
加载并配置漏洞模块
use exploit/windows/smb/ms17_010_eternalblue
设置参数
配置目标IP等参数
set RHOSTS target_ip

Python脚本编写

requests库
发送HTTP请求
import requests; r = requests.get(url)
re正则表达式
文本匹配和提取
import re; re.search(pattern, text)
subprocess模块
执行系统命令
subprocess.run(['ls', '-l'])
paramiko
SSH连接
SSH客户端库

Shell脚本

变量定义
VAR="value"
条件判断
if [ condition ]; then ... fi
循环
for i in $(seq 1 10); do ... done
函数
function name() { ... }
