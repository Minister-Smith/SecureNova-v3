第三阶段：Web基础

Phase 03

目标：理解Web工作原理，掌握基础开发技能

HTTP请求方法

GET
请求获取资源，参数在URL中
GET /api/users?id=1
POST
提交数据到服务器，参数在请求体中
POST /api/login
PUT
更新资源，替换整个资源
PUT /api/users/1
DELETE
删除指定资源
DELETE /api/users/1

HTTP状态码

1xx - 信息性
请求已接收，继续处理
100 Continue
2xx - 成功
请求成功处理
200 OK, 201 Created
3xx - 重定向
需要进一步操作
301 Moved Permanently
4xx - 客户端错误
请求有问题
404 Not Found, 403 Forbidden
5xx - 服务器错误
服务器处理失败
500 Internal Server Error

HTTP头字段

Host
请求的主机名和端口
User-Agent
客户端浏览器信息
Referer
来源页面URL
Cookie
客户端存储的会话信息
Content-Type
请求体的MIME类型

Session与Cookie

Cookie
存储在客户端浏览器的小型数据
key=value; expires=date
Session
存储在服务器端的会话数据
通过Session ID关联
区别
Cookie在客户端，Session在服务器端

HTTPS原理

SSL/TLS握手
客户端与服务器建立安全连接
证书验证
验证服务器身份真实性
对称加密
数据传输使用对称密钥加密
非对称加密
交换对称密钥时使用

HTML基础

HTML结构
<!DOCTYPE>, <html>, <head>, <body>
常用标签
<div>, <p>, <a>, <form>
表单元素
<input>, <select>, <textarea>

JavaScript基础

变量声明
var, let, const
数据类型
string, number, boolean, object, array
函数
function, arrow function
DOM操作
getElementById, querySelector

Python基础

变量与数据类型
int, str, list, dict, tuple
条件语句
if, elif, else
循环
for, while
函数定义
def function_name(): ...

Flask框架

路由定义
@app.route('/path')
请求处理
request对象获取参数
模板渲染
render_template('index.html')
静态文件
static文件夹存放CSS/JS

SQL基础

SELECT查询
SELECT * FROM users WHERE id=1
INSERT插入
INSERT INTO users(name) VALUES('test')
UPDATE更新
UPDATE users SET name='new' WHERE id=1
DELETE删除
DELETE FROM users WHERE id=1
